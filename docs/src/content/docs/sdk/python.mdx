The Python SDK is the async client for Python agents. It uses Pydantic models and exposes the same platform concepts as the TypeScript SDK: identity, authorization, MCP invocation, Airport discovery, HITL, skills, and transaction receipts.

## Install

```bash
pip install agentid-sdk
```

Framework extras:

```bash
pip install "agentid-sdk[langchain]"
pip install "agentid-sdk[crewai]"
pip install "agentid-sdk[autogen]"
pip install "agentid-sdk[dev]"
```

Requirements:

- Python 3.10 or newer.
- `httpx`, Pydantic 2, and PyNaCl.
- A running eyeVesa gateway.

## Minimal Client

```python
import asyncio
from agentid_sdk import AgentClient, AgentConfig

async def main():
    config = AgentConfig(
        agent_id="support-agent-001",
        name="support-agent",
        owner="customer-success",
        gateway_endpoint="http://localhost:9443",
    )

    async with AgentClient(config) as client:
        await client.connect()
        print(client.agent_id)
        print(client.trust_score)

asyncio.run(main())
```

## Invoke A Resource

<Steps>
1. Connect the agent.
2. Discover MCP resources.
3. Select a resource ID.
4. Invoke a tool. The gateway evaluates policy before MCP execution.
</Steps>

```python
async with AgentClient(config) as client:
    await client.connect()
    tools = await client.discover("mcp")

    weather_tool = next(
        (tool for tool in tools if "weather" in tool.name),
        None,
    )
    if weather_tool is None:
        raise RuntimeError("No weather MCP resource is registered")

    result = await client.invoke(
        resource_id=weather_tool.resource_id,
        tool="read",
        params={"location": "Kuala Lumpur"},
    )

    print(result.success)
    print(result.trust_score)
    print(result.data)
```

## Core API

| Method | Returns | Purpose |
| --- | --- | --- |
| `connect()` | `AgentClient` | Register agent, login, and update trust state. |
| `discover(capability)` | `list[ToolInfo]` | Discover registered resources by capability. |
| `invoke(resource_id, tool, params)` | `InvokeResult` | Authorize then execute an MCP tool. |
| `delegate(delegatee_id, scope, reason)` | `DelegateResult` | Delegate scoped permissions. |
| `mcp_initialize()` | `McpCapabilities` | Start direct MCP negotiation. |
| `mcp_list_tools()` | `list[McpTool]` | List available MCP tools. |
| `mcp_call_tool(tool_name, arguments)` | `dict` | Call a named MCP tool directly. |
| `verify_signature(agent_id, message, signature)` | `bool` | Verify Ed25519 signatures. |

## Pydantic Models

```python
class AgentConfig(BaseModel):
    agent_id: str
    name: str
    owner: str
    gateway_endpoint: str

class InvokeResult(BaseModel):
    success: bool
    data: dict[str, Any]
    trust_score: float

class AuthorizeResult(BaseModel):
    allowed: bool
    requires_hitl: bool
    reason: str
    trust_delta: float
```

## Error Handling

```python
from agentid_sdk import HitlRequiredError, NotAuthorizedError

try:
    await client.invoke(resource_id, "delete", {"id": "invoice-123"})
except HitlRequiredError as error:
    print(f"Human approval required: {error}")
except NotAuthorizedError as error:
    print(f"Policy denied: {error}")
```

See [Errors](/sdk/errors/) for the full exception hierarchy.
